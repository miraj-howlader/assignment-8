My project is a modern Next.js “Tiles Gallery” web app where users can browse, search, and view detailed information about different tile designs. It uses a clean grid layout with reusable components like TilesCard and dynamic routing for individual tile pages. I also added a live search feature that filters tiles instantly, making the UI more interactive and user-friendly.

👌👌 In This project i used the popular framework Nextjs and some helper ui libary like
** Hero ui
** Authentication for Better-auth
** react-hot-toast
** framer motion
** react-icons lucide-react
